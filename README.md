# ft_printf
A simple reimplementation of the famous printf function. 

## Features

Flags accepted:
-  ( - )  ( 0 )  ( . )  ( * )

Conversions covered:
- ( c )  ( s )  ( p )  ( d )  ( i )  ( u )  ( x )  ( X )  ( % )

Example of usage:
```
ft_printf("Hello %s! %d is a great number.\n", "stranger", 42);
```
Prints:
```
Hello stranger! 42 is a great number.
```
